Domain Names and ICANN

ICANN is an acronym for Internet Corporation for Assigned Names and Numbers. They are a nonprofit corporation that oversees internet related tasks, such as the assignment of both IP address and domain names.

Learn More

IP Addresses, Packets and Routing

A computer’s address is referred to as the IP Address, short for Internet Protocol Addresses. IP addresses are a string of digits, similar to a phone number, and they allow devices to communicate with each other.

Learn More

DNS

DNS is an acronym for Domain Name System. DNS is the protocol that connects URLs with their IP addresses. DNS allows people to search for websites using familiar names, such as nike.com, rather using their IP addresses.

Learn More






HTTP and HTTPS protocols

HTTP is an acronym for Hypertext Transfer Protocol. In laymen terms, HTTP can be thought of as the language used to communicate between web browsers and servers. HTTPS is a more advanced version of HTTP that has encryption.

Learn More

W3C and HTML and CSS

Created in 1994, the World Wide Web Consortium (W3C) is an international community where Member organizations, full-time staff, and the public work together to develop Web standards. The W3C develop HTML and CSS standards that allow for websites to look and work the same in different web browsers.
